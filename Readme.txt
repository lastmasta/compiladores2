Proyecto desarrollado por Rodrigo Pineda como parte del curso de Organización de Lenguajes y Compiladores 2 de la facultad de ingeniería de la Universidad de San Carlos de Guatemala.

El proyecto está escrito en C++ utilizando el IDE QtCreator

Dependencias:
Se debe tomar en cuenta que para poder hacer uso del software se debe contar con las siguientes dependencias:

- Bison: generador de analizadores sintácticos para gramáticas LALR 
Instalacion en Debian y derivados: apt-get install bison

- Flex: generador de analizadores léxicos
Instalacion en Debian y derivados: apt-get install flex

- Graphviz: generador de grafos
Instalacion en Debian y derivados: apt-get install graphviz
